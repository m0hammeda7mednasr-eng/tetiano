# 🔧 حل خطأ 500 - Shopify Connect

## 📊 الوضع الحالي

### ✅ ما تم إنجازه
- Railway نشر الكود الجديد بنجاح
- المسارات `/api/app/*` تعمل الآن (لم تعد 404)

### ❌ المشكلة الجديدة
```
POST /api/app/shopify/connect → 500 Internal Server Error
```

**السبب**: جدول `shopify_oauth_states` غير موجود في قاعدة البيانات

---

## 🔍 التشخيص

عندما تحاول الـ Frontend الاتصال بـ Shopify، الـ Backend يحاول الكتابة في جدول `shopify_oauth_states` لحفظ حالة OAuth، لكن الجدول غير موجود.

### الكود الذي يسبب الخطأ:
```typescript
// backend/src/routes/app.ts:1175
const stateInsert = await supabase.from("shopify_oauth_states").insert({
  state,
  shop: shopDomain,
  brand_id: storeId,
  user_id: req.user?.id || null,
  api_key: apiKey,
  api_secret: apiSecret,
  expires_at: new Date(Date.now() + 15 * 60 * 1000).toISOString(),
});
```

## ✅ الحل

### الخطوة 1: تشغيل SQL Script

1. افتح [Supabase Dashboard](https://supabase.com/dashboard)
2. اختر مشروعك
3. اذهب إلى **SQL Editor**
4. انسخ محتوى الملف `COMPLETE_DATABASE_FIX.sql`
5. الصقه في SQL Editor
6. اضغط **Run**

### الخطوة 2: التحقق من النجاح

بعد تشغيل السكريبت، يجب أن ترى:
```
Database fix completed successfully!
```

## 📋 ما يفعله السكريبت

السكريبت ينشئ الجداول التالية (إذا لم تكن موجودة):

### 1. shopify_oauth_states
لحفظ حالة OAuth عند الاتصال بـ Shopify

### 2. shopify_sync_runs

### 3. report_attachments

### 4. report_comments
لحفظ تعليقات على التقارير

### 5. reports
لحفظ التقارير اليومية

### 6. shopify_customers
لحفظ بيانات العملاء من Shopify

## 🧪 اختبار بعد الإصلاح

2. اذهب إلى Settings → Shopify

1. افتح Railway Dashboard

## 🔍 استكشاف الأخطاء

### خطأ: "relation shopify_oauth_states does not exist"

**الحل**: 
- تأكد من تشغيل `COMPLETE_DATABASE_FIX.sql`
- تحقق من أن السكريبت نجح بدون أخطاء

### خطأ: "permission denied for table shopify_oauth_states"

**الحل**:
- السكريبت يمنح الصلاحيات تلقائياً
- إذا استمر الخطأ، شغّل:
```sql
GRANT ALL ON shopify_oauth_states TO authenticated, anon;
```

### خطأ: "column brand_id does not exist"

**الحل**:
- السكريبت يضيف الأعمدة المطلوبة
- تحقق من أن السكريبت اكتمل بنجاح

## 📊 الجداول المطلوبة - Checklist

بعد تشغيل السكريبت، تحقق من وجود هذه الجداول:

```sql
-- تشغيل هذا في SQL Editor للتحقق
SELECT table_name 
FROM information_schema.tables 
WHERE table_schema = 'public' 
  AND table_name IN (
    'shopify_oauth_states',
    'shopify_sync_runs',
    'report_attachments',
    'report_comments',
    'reports',
    'shopify_customers',
    'stores',
    'store_memberships',
    'shopify_connections',
    'user_profiles',
    'products',
    'variants',
    'inventory_levels',
    'shopify_orders'
  )
ORDER BY table_name;
```

**المتوقع**: يجب أن ترى جميع الجداول في القائمة
